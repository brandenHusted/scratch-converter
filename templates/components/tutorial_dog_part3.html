Take your zip file and unzip it by right-clicking the zip file and clicking extract here.




Remember when you save your project on your computer it could be helpful for youself to make a seperate folder so you can find your project easily and quickly.


Downloading Python

If you have not downloaded python before open Visual Studio and on the left side of your screen you should see a tab that is called extensions.


Click on extensions and search for python and download the extension shown below.





Navagate to your folder

Go to Visual Studio and click file and Open Folder and open your unzipped folder.


In the top left of your screen click terminal in visual studio.


In the terminal navigate to your folder with your unzipped code by using the command cd
